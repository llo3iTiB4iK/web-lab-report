Структура проєкту

Проєкт складається з семи HTML-сторінок

  1. index.html – Головна сторінка застосунку
  2. cars.html – Сторінка з переліком усіх автомобілів, що пропонуються до прокату
  3. car.html – Сторінка окремого авто, що пропонується до прокату
  4. rules.html – Сторінка з правилами та умовами прокату
  5. about_us.html – Сторінка з контактами та додатковою інформацією про компанію, що надає авто в прокат
  6. login.html – Сторінка входу для персоналу сервісу прокату
  7. administration.html – Сторінка управління переліком автомобілів, доступних для прокату

Загальна структура кожної сторінки така:

    <html>
        <head>
            ...
        </head>

        <body>
            <header>
                ...
                <nav>
                    ...
                </nav>
            </header>

            <main>
                ...
            </main>

            <footer>
                ...
            </footer>
        </body>
    </html>

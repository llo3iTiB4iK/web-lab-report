Для надання нашому WEB застосунку необхідного вигляду, були застосовані такі селектори класів:

  1. Вирівнювання тексту за замовчуванням для елементів сторінки в нас задане по центру, але є елементи, що потребують вирівнювання тексту по лівому краю:

    .left_aligned {
    text-align: left;
}

    Рисунок 1 – Приклад елементу з класом "left_aligned"

  2. Для кращого візуального сприйняття, створимо клас, який буде для елементів-зображень давати ефект тіні:

    img.shadowed {
    border:.5px solid;
    box-shadow: rgba(0, 0, 0, 0.25) 0 54px 55px, rgba(0, 0, 0, 0.12) 0 -12px 30px, rgba(0, 0, 0, 0.12) 0 4px 6px,
            rgba(0, 0, 0, 0.17) 0 12px 13px, rgba(0, 0, 0, 0.09) 0 -3px 5px;

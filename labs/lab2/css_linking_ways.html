Існує три типи підключення стилів, кожен з яких має різну реалізацію та використовується в різних випадках:

  • Вбудований стиль (inline) - використовується для застосування унікального стилю для окремого HTML-елемента. Реалізується за допомогою додавання атрибута style до бажаного елемента з CSS-правилами у лапках як значенням.
    Приклад: 
        <!DOCTYPE html>
    <html>
        <body>
            <h1 style="color:blue;text-align:center;">Це заголовок</h1>
            <p style="color:red;">Це параграф.</p>
        </body>
    </html>
  • Внутрішня таблиця стилів (internal) - використовується, якщо окрема сторінка має свій стиль, який не застосовується ніде на інших сторінках. Реалізується за допомогою визначення css-правил у тегу <style> всередині розділу <head> HTML сторінки.
    Приклад: 
        <!DOCTYPE html>
    <html>
        <head>
            <style>
                body {
                  background-color: linen;
                }
                h1 {
                  color: maroon;
                  margin-left: 40px;
                }
            </style>
        </head>
        <body>
            <h1>Це заголовок</h1>
            <p>Це параграф.</p>
        </body>
    </html>
  • Зовнішня таблиця стилів (external) - використовується, якщо багато сторінок мають однакові стилі. Реалізується в окремому файлі, де просто треба визначити css-правила та підключити його до кожної сторінки, де треба застосувати ці стилі за допомогою тега <link> у розділі <head>.
    Приклад: 
    HTML сторінка:
    <!DOCTYPE html>
    <html>
        <head>
            <link rel="stylesheet" href="styles.css">
        </head>
        <body>
            <h1>Це заголовок</h1>
            <p>Це параграф.</p>
        </body>
    </html>

styles.css:
    body {
      background-color: lightblue;
    }
    h1 {
      color: navy;
      margin-left: 20px;
    }

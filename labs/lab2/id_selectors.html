Для надання нашому WEB застосунку необхідного вигляду, були застосовані такі селектори ідентифікаторів:

  1. Я підготував три типи фонів, для застосування різних фонів до різних сторінок, для тега body на кожній сторінці постановлюємо різні id та застосуємо наступні правила:

    #main_page {
    background-image: url("img/bg1.jpeg");
}
#auto_list_page, #car_page, #login_page {
    background-image: url("img/bg2.jpg");
}
#rules_page {
    background-color: #f1e6c5;
}
#about_us_page, #admin_page {
    background-image: url("img/bg3.jpg");
}

    Рисунок 1 – Головна сторінка з встановленим фоном

  2. В навігаційній панелі, для посилання на сторінку, на якій ми і так знаходимось зараз, додамо ідентифікатор, за яким даватимемо посиланню унікальний вигляд:

    #link_to_current_page {
    background-color: #257929;
}

    Рисунок 2 – Посилання на сторінку переліку авто на сторінці переліку авто має свій вигляд

  3. Для панелі фільтрів на сторінці переліку авто налаштуємо візуальний вигляд:

    #filters {
    margin: 0 auto 50px;
    border-collapse: collapse;
    table-layout: fixed;
    width: 75%;
    background-color: white;
}

    Рисунок 3 – Панель фільтрів з ідентифікатором "filters"

  4. Для списка правил застосуємо унікальні стилі:

    #rules_list {
    background-color: #FF8D8D;
    margin: auto 2%;
    padding:.5% 2% .5% 3%;
}

    Рисунок 4 – Приклад списку з ідентифікатором "rules_list"

  5. Для таблиці з переліком послуг, що включаються в ціну прокату, застосуємо унікальні стилі:

    #price_includes {
    margin: 50px auto 0;
}

    Рисунок 5 – Приклад таблиці з ідентифікатором "price_includes"

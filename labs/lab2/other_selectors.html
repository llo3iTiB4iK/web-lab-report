Для надання нашому WEB застосунку необхідного вигляду, були застосовані різноманітні інші селектори та комбінації:

  1. Універсальний селектор для задання алгоритму розрахунку розмірів блоків:

    * {
    box-sizing: border-box;
}

  2. Дочірній селектор для зміни стиля параграфу тексту, який іде на першому рівні вкладеності тега main:

    main>p {
    background-color: rgba(255,255,255,0.8);
    margin-left: 20px;
    margin-right: 20px;
    padding: 5px;
}

    Рисунок 1 – Параграф тексту на першому рівні вкладності від main

  3. Комбінація селектора нащадка та псевдокласу first-child для зміни стилю клітинки, яка є першою серед дітей батьківського блока:

    #price_includes td:first-child {
    width: 55%;
}

    Рисунок 2 – Змінений стиль для клітинки, яка є першою дитиною батьківського блока (рядка таблиці)

  4. Застосування комбінації селектора нащадків, селектора дочірнього елемента та псевдокласу before, для встановлення іншого маркера для маркованого списку в елементі з ідентифікатором price_includes:

    #price_includes ul>li:before{
    content: '✔';
    color: green;
    margin-right: 20px;
}

    Рисунок 3 – Змінений стиль маркера списку

  5. Застосування селектора атрибуту для кнопки, що здійснює відправку форми входу:

    #login_page form input[type="submit"] {
    width: 34%;
    margin: 0 33%;
    background-color: #409F29;
    color: white;
    font-size: larger;
    border-radius: 10px;
}

    Рисунок 4 – Вигляд кнопки відправки форми

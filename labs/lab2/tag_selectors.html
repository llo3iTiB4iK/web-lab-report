Для надання нашому WEB застосунку необхідного вигляду, були застосовані різноманітні селектори тегів:

  1. Загальні налаштування для тега body:

    body {
    text-align: center;
    margin: 0;
    min-height: 100%;
    display: flex;
    flex-direction: column;
}

  2. Налаштування для заголовка сторінки, щоб він був прикріплений до верху сторінки при прокручуванні:

    header {
    position:sticky;
    top:0;
    z-index:2;
}

    Рисунок 1 – Заголовок сайту до прокручування

    Рисунок 2 – Заголовок сайту після прокручування

  3. Налаштування кольору фону та тексту, відступів та шрифту для навігаційної панелі:

    nav {
    padding:10px;
    background-color: rgba(0,0,0,0.7);
    color:white;
    margin-bottom: 10px;
    font: 24px "Courier New", sans-serif;
}

    Рисунок 3 – Навігаційна панель сайту

  4. Налаштування верстки та шрифту для головного вмісту сторінки:

    main {
    font: 18px "Trebuchet MS", serif;
    flex: 1;
    display: flex;
    flex-direction: column;
}

    Рисунок 4 – Вигляд головного вмісту сторінки

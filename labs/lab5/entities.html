JavaScript entity - це спеціальний фрагмент коду JavaScript, який може бути вставлений безпосередньо як значення будь-якого атрибута HTML у документі. Оскільки це код JavaScript, значення не обов'язково є статичним і може змінюватися на ходу згідно з маніпуляційним скриптом. Синтаксис JavaScript entity виглядає так: 

   &{ JavaScript-вказівки };

Нижче показано два HTML-теги: перший використовує звичайне HTML-значення для свого атрибута, другий - значення JavaScript entity:

Звичайне HTML-значення:
        <body background="image1.gif">
Значення JavaScript entity:
        <body background="&{JavaScript-statements };">

Особливістю у другому випадку є те, що значення атрибута HTML тепер є не статичним, а динамічним - його значенням можна маніпулювати за допомогою JavaScript.

Всередину entity може бути вбудований будь-який оператор JavaScript або змінна. Ось простий приклад використання JavaScript entities для призначення різного шляху до зображення залежно від часу доби: 

<script>
    let my_image = "day.gif";
    const today = new Date();
    const hour = today.getHours();
    if (hour > 12)
        my_image="night.gif";
</script>

<img src="&{my_image};">

Таким чином, при завантаженні сторінки враховується час, і якщо година в поточний момент більше за 12 - показується зображення night.gif, а якщо ні - то day.gif.

JavaScript дозволяє реагувати на різні події, які виникають у веб-браузері користувача. Обробники подій - це функції, які виконуються відповідно до специфічних дій користувача або виникнення певних подій на веб-сторінці. Ось декілька ключових понять і прикладів для розуміння роботи з обробниками подій:

  1. Додавання обробників подій:

    document.getElementById('myButton').addEventListener('click', function() {
    window.alert('Кнопка була натиснута!');
});

    Тепер при натисканні на кнопку з ідентифікатором myButton буде з'являтись повідомлення про її натискання.

  2. Типи подій (найчастіше використовувані):

    • 'click' : Виникає при натисканні кнопки миші.
    • 'change' : Виникає при зміні HTML-елемента.
    • 'mouseover' : Виникає при наведенні курсора на елемент.
    • 'mouseout' : Виникає при виведенні курсора з елемента.
    • 'keydown' : Виникає при натисканні кнопки на клавіатурі.
    • 'load' : Виникає при завершенні завантаження сторінки.

  3. Видалення обробників подій:

    const button = document.getElementById('myButton');
const clickHandler = function() {
    console.log('Кнопка була натиснута!');
};
button.addEventListener('click', clickHandler);
button.removeEventListener('click', clickHandler);

    Якщо зберегти посилання на функцію-обробник події, то пізніше можна її видалити за допомогою removeEventListener.

Під час виконання обробника події, об'єкт події передається як аргумент функції обробника. Це дозволяє отримати доступ до додаткових властивостей та інформації про подію.

Тег <script> використовується для вставки JS-скриптів на веб-сторінку. Це дає можливість динамічної зміни вмісту та поведінки сторінки. Ось кілька способів, якими можна використовувати тег <script>:

  • Вкладення в HTML

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Вставка скриптів</title>
</head>
<body>
    <h1>Вставка скриптів на веб-сторінку</h1>
    <script>
        // JavaScript код тут
        console.log("Це скрипт, вбудований безпосередньо в HTML");
    </script>
</body>
</html>

  • Зовнішні файли скриптів

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Вставка скриптів</title>
    <script src="script.js"></script>
    <!-- Скрипт виноситься в окремий файл script.js -->
</head>
<body>
    <h1>Вставка скриптів на веб-сторінку</h1>
</body>
</html>

  • Динамічна вставка через JavaScript

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Вставка скриптів</title>
</head>
<body>
    <h1>Вставка скриптів на веб-сторінку</h1>
    <div id="scriptContainer"></div>
    <script>
        // Створення нового тегу script і вставка його на сторінку
        let script = document.createElement("script");
        script.src = "script.js";
        document.getElementById("scriptContainer").appendChild(script);
    </script>
</body>
</html>

Вибір конкретного методу вставки скриптів залежить від потреб та структури проєкту. Також, варто звернути увагу на правильне розміщення скриптів для досягнення оптимальної продуктивності та сумісності з різними браузерами.

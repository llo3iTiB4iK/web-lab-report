При виборі того чи іншого гіпертекстового посилання викликається програма реалізації гіпертекстового переходу. У WEB-технології стандартною програмою є програма завантаження сторінки. JavaScript дозволяє змінити стандартну програму на програму користувача. Для того щоб відрізнити стандартний перехід по протоколу HTTP від переходу, запрограмованого JavaScript, розробники мови ввели нову схему URL - JavaScript: 

  <a href="JavaScript:JavaScript_код">...</a>

  <img src="JavaScript:JavaScript_код" alt="...">

В даному випадку "JavaScript_код" це програми-обробники на JavaScript, які викликаються при виборі гіпертекстового посилання в першому випадку та при завантаженні картинки - у другому. При програмуванні гіпертекстового переходу інтерпретатор отримує управління після того, як викликався перехід за гіпертекстовим посиланням.

Наприклад, при натисканні на кнопку типу submit у формі можна заповнити текстове поле цієї ж форми: 

  <form name=form method=post
      action="JavaScript:window.document.form.i.value='Ви натиснули кнопку'; void(0);">
    <input type="text" name="i">
    <input type="submit" value="submit">
  </form>

Звичайно, я навів найпростіші приклади використання, але в url можна розміщувати й складні програми та функції.
